La funcion base de esta aplicacion es comprobar el uso de celery para las tareas asincrona en segundo planos. El formulario de arriba permite numeros entre 50 y 500 que serviran para determinar el numeros de usuarios a crear.

Para iniciar el proceso ejecutamos en la terminal donde tenemos ubicado el proyecto

celery -A core worker -l info
